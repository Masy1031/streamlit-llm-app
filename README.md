# Streamlit LLM App

## 概要
Streamlit LLM Appは、AIを活用してさまざまな専門家として振る舞うアシスタントと対話できるアプリケーションです。ユーザーは専門家の種類を選択し、質問を入力することで、選択した専門分野に応じた回答を得ることができます。

## 主な機能
- **専門家の種類を選択**: 「世界史」「物理学」「国際料理」の3つの専門家から選択可能。
- **質問入力**: ユーザーが自由に質問を入力可能。
- **AIによる回答**: 選択した専門家に応じた回答を生成。
- **デバッグ情報の表示**: 選択された専門家とシステムメッセージを確認可能。

## 使用技術
- **Python**: アプリケーションのロジックを構築。
- **Streamlit**: ユーザーインターフェースを提供。
- **LangChain**: LLM（大規模言語モデル）との連携を実現。

## インストールと実行方法

### 必要条件
- Python 3.8以上
- 仮想環境（推奨）

### 手順
1. リポジトリをクローンします。
   ```bash
   git clone https://github.com/Masy1031/streamlit-llm-app.git
   ```
2. ディレクトリに移動します。
   ```bash
   cd streamlit-llm-app
   ```
3. 仮想環境を作成して有効化します。
   ```bash
   python -m venv env
   # Windows
   .\env\Scripts\activate
   # macOS/Linux
   source env/bin/activate
   ```
4. 必要なパッケージをインストールします。
   ```bash
   pip install -r requirements.txt
   ```
5. アプリケーションを実行します。
   ```bash
   streamlit run app.py
   ```

## 使い方
1. アプリを起動すると、専門家の種類を選択するラジオボタンが表示されます。
2. 質問を入力し、「送信」ボタンを押します。
3. 選択した専門家に応じた回答が表示されます。

## デバッグ機能
- 選択された専門家の種類とシステムメッセージが画面に表示されるため、設定が正しく反映されているか確認できます。
